## keybinds

| key | action |
|-----|--------|
| `Ctrl+N` | create new note |
| `Ctrl+L` | browse notes |
| `Ctrl+S` | save & close |
| `Ctrl+D` | delete note |
| `Esc` | go back |
| `Ctrl+C` | quit |
